4.1.1功能概述

船公司提供空白提单供我司打印正本提单，我司对空白提单进行交接入库、使用管理，可查看空白提单详细打印、作废情况，按船司要求回收作废提单纸；放单后打印正本提单时将序列号与提单号绑定。

4.1.2状态说明

使用状态：已入库、已使用、已归档

4.1.5功能点描述

1.【查询】

  用户输入查询条件，查询出符合条件的空白提单序列号段数据，用列表形式展示

2. 【重置】

  清空查询条件

3. 【新增】

  新增一条空白提单序列号段数据，编号起始、结束字段必填，默认“已入库”状态， 根据编号段自动计算总数量

4. 【编辑】

  编辑当前选中的空白提单序列号段数据

5. 【作废】

   作废当前选中的空白提单序列号段数据，该号码段内的序列号不可用

6. 【导出作废清单】

   导出当前选中的空白提单序列号段内作废的序列号清单，用于与船司线下交接纸质单据

【子页面-作废】

作废选中的空白提单序列号，选择作废原因

【子页面-回收】

回收选中的空白提单序列号，清空该序列号关联的提单号，释放当前序列号可再次打印
